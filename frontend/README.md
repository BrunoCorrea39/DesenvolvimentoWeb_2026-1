# ProjetAí Frontend

Interface React do ProjetAí.

## Requisitos

- Node `20.19+` ou `22.12+`
- Backend rodando em `http://127.0.0.1:8000`

## Comandos

```powershell
npm install
npm run dev
```

Build:

```powershell
npm run build
```

Se a API estiver em outra URL, configure:

```env
VITE_API_URL=http://127.0.0.1:8000/api
```
